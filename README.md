# ✅ Todo App

A simple and interactive **Todo App** built using **HTML, CSS, and JavaScript**. This application allows users to add and delete tasks, with all task data stored in JavaScript variables during the current session.

## 📌 Project Overview

The Todo App helps users manage their daily tasks efficiently through a clean and user-friendly interface. Tasks are stored in a JavaScript array and dynamically rendered on the webpage.

> Note: Since the data is stored in variables, all tasks are cleared when the page is refreshed.

## ✨ Features

* Add new tasks
* Delete existing tasks
* Dynamic task rendering
* Clean and responsive UI
* Input validation for empty tasks
* Lightweight and easy to use

## 🛠️ Technologies Used

* **HTML5** – Structure and content
* **CSS3** – Styling and layout
* **JavaScript (ES6)** – Functionality and task management


## 🚀 Getting Started

### Prerequisites

You only need a modern web browser.

### Installation

1. Clone the repository:

```bash
git clone https://github.com/umermemon1/Todo-Del-Edit.git
```

2. Navigate to the project directory:

```bash
cd Todo-Del-Edit
```

3. Open `index.html` in your browser.

## ⚙️ How It Works

1. Enter a task in the input field.
2. Click the **Add** button.
3. The task is stored in a JavaScript array.
4. Tasks are displayed dynamically on the page.
5. Click **Delete** to remove a task from the list.
